  UCC5304 WIRING DETAIL (repeat for U1-U4)
  ==========================================

                    UCC5304 (SOIC-6)
                   +-----+--+-----+
           VCC1 ---|1  VCC1  6|--- VCC2 (bootstrap/gate supply)
                   |         |
     RP2040 GPx ---|2  INA  5|--- OUTB (gate drive output)
                   |         |
           GND1 ---|3  GND1 4|--- GND2
                   +---------+

  Input side (pins 1-3): 3.3V domain
    - VCC1 → 3.3V from RP2040
    - GND1 → RP2040 GND
    - INA  → RP2040 GPIO (GP2/3/4/5)

  Output side (pins 4-6): gate drive domain
    - VCC2 → gate supply (bootstrap cap for high-side, direct for low-side)
    - GND2 → source of driven MOSFET
    - OUTB → gate of driven MOSFET

  Mapping:
    U1: GP2 → INA → P1 high-side MOSFET gate
    U2: GP3 → INA → P2 high-side MOSFET gate
    U3: GP4 → INA → N1 low-side MOSFET gate
    U4: GP5 → INA → N2 low-side MOSFET gate


  INA226 I2C CURRENT SENSING
  ============================

  All 4 INA226 modules share one I2C bus on GP6 (SDA) / GP7 (SCL).
  Each module measures current through one leg of the H-bridge.

  I2C Bus:
                     4.7k         4.7k
    3.3V ───────────┤────────────┤
                    |            |
    GP6 (SDA) ──────●────┬───┬───┬───┬──
    GP7 (SCL) ──────●────┤   |   |   |
                         |   |   |   |
                      +--+--++--++--++--+
                      |INA226|INA226|INA226|INA226|
                      | P1   | P2   | N1   | N2   |
                      |0x40  |0x41  |0x44  |0x45  |
                      +------+------+------+------+

  Address Configuration (D-FLIFE breakout modules):
  ┌────────┬─────┬─────┬─────────┬──────────────────┐
  │ Module │  A1 │  A0 │ Address │ Position         │
  ├────────┼─────┼─────┼─────────┼──────────────────┤
  │ P1     │ GND │ GND │  0x40   │ High-side left   │
  │ P2     │ GND │ VS  │  0x41   │ High-side right  │
  │ N1     │ VS  │ GND │  0x44   │ Low-side left    │
  │ N2     │ VS  │ VS  │  0x45   │ Low-side right   │
  └────────┴─────┴─────┴─────────┴──────────────────┘

  INA226 Shunt Resistor Placement:

                +HV
                 |
    ┌──[Rshunt]──●── P1 MOSFET drain
    │  INA226    |
    │  Vin+/Vin- |
    │  (P1,0x40) |
                 ●── P1 MOSFET source ──── Node A
                 |
    ┌──[Rshunt]──●── N1 MOSFET drain ──── Node A
    │  INA226    |
    │  Vin+/Vin- |
    │  (N1,0x44) |
                 ●── N1 MOSFET source
                 |
                GND

  (Mirror for P2/N2 on the right leg)


  PERIPHERALS
  ============

  Rotary Encoder:
    GP8  → CLK
    GP9  → DT
    GP10 → SW (active low, internal pull-up)
    GND  → GND
    VCC  → 3.3V

  TM1637 7-Segment Display:
    GP14 → CLK
    GP15 → DIO
    GND  → GND
    VCC  → 3.3V (or 5V, check module)

  ADC Inputs (future transient capture):
    GP26 → ADC ch0
    GP27 → ADC ch1
    GP28 → ADC ch2


  POWER RAILS
  =============

  3.3V (RP2040 regulated output):
    → UCC5304 VCC1 (all 4 drivers, input side)
    → INA226 VDD (all 4 modules)
    → I2C pull-ups
    → Rotary encoder VCC
    → TM1637 (if 3.3V tolerant)

  GND (common):
    → RP2040 GND
    → UCC5304 GND1 (all 4 drivers, input side)
    → INA226 GND (all 4 modules)
    → Encoder GND
    → TM1637 GND
    → Battery negative (power GND)

  !! IMPORTANT: UCC5304 output-side GND2 connects to the MOSFET SOURCE,
     NOT to the common ground (especially for high-side drivers).


  WIRING ORDER (recommended)
  ============================

  1. RP2040 USB → Pi (verify /dev/ttyACM0 appears)
  2. GP2-5 → UCC5304 U1-U4 inputs
     → Test with TUI debug mode (g, then 1-4) before proceeding
  3. UCC5304 outputs → MOSFET gates
     → Verify each FET switches correctly
  4. GP6/GP7 → I2C bus with pull-ups
  5. INA226 modules → I2C bus, set address jumpers
     → Verify with i2cdetect: should see 0x40, 0x41, 0x44, 0x45
  6. INA226 shunt resistors inline with each H-bridge leg
  7. Rotary encoder + display (GP8-10, GP14-15)


  H-BRIDGE STATE TRUTH TABLE (reference)
  ========================================

  State │ P1  P2  N1  N2 │ Current Path    │ Description
  ──────┼─────────────────┼─────────────────┼────────────
    0   │ ON  off ON  off │ +HV→P1→A→N1→GND│ Forward
    1   │ ON  off off ON  │ +HV→P1→A→B→N2  │ Cross A→B
    2   │ off ON  ON  off │ +HV→P2→B→A→N1  │ Cross B→A
    3   │ off ON  off ON  │ +HV→P2→B→N2→GND│ Reverse
    4   │ ON  ON  ON  ON  │ All paths       │ Discharge
    5   │ off off off off │ None            │ Idle
